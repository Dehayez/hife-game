# NotFound Component

404 page component with Hife background and logo.

## Usage

```jsx
import { NotFound } from './components/NotFound';

// Use in React app
<NotFound />
```

## Features

- Hife-themed background with gradient animation
- Floating logo animation
- Responsive design
- "Return to Game" button to navigate home
- Optimized animations and transitions

## Structure

- `NotFound.jsx` - Main component
- `NotFound.scss` - Component styles (BEM structure)
- `index.js` - Component export

## Styling

Uses CSS custom properties from the main theme:
- `--color-bg-primary`
- `--color-accent`
- `--color-character-lucy`
- `--color-text-primary`

# Game Mode Images

This directory contains AI-generated images for each game mode button.

## Required Images

Place the following images in this directory:

- `free-play.png` - Image for "Forest Wander" mode
- `time-trial.png` - Image for "Crystal Shrine" mode  
- `collection.png` - Image for "Gem Gathering" mode
- `survival.png` - Image for "Shadow Escape" mode
- `shooting.png` - Image for "Mystic Battle" mode

## Image Specifications

- **Recommended size**: 160x160px or larger (will be displayed at 80x80px)
- **Format**: PNG with transparency (preferred) or JPG
- **Style**: Should match the game's magical forest aesthetic
- **Content**: Should visually represent each game mode's theme

## Notes

- Images will automatically hide if they fail to load
- Images are displayed above the mode name in the game mode selector
- Images use `object-fit: cover` to maintain aspect ratio

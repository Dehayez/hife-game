# Arena Images

This directory contains images for arena selection in the game UI.

## Required Images

- `forest-plaza.png` - Image for Forest Plaza (20x20) arena
- `ancient-grove.png` - Image for Ancient Grove (40x40) arena

## Image Specifications

- **Format**: PNG
- **Recommended Size**: 400x300px or similar aspect ratio (4:3)
- **Style**: Should match the game's magical forest aesthetic
- **Content**: 
  - Forest Plaza: A cozy forest clearing with magical elements
  - Ancient Grove: A vast ancient forest with mystical atmosphere

## Image Generation

These images should be AI-generated to match the game's visual style. Consider using prompts like:

- **Forest Plaza**: "A cozy magical forest clearing, 20x20 area, moonlit, glowing mushrooms, fireflies, warm atmosphere, game art style"
- **Ancient Grove**: "A vast ancient magical forest, 40x40 area, mystical atmosphere, towering trees, magical elements, epic scale, game art style"
